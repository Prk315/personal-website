Background Layer Test

This demonstrates the layered background effect with:

Matte white base → Vignette darkening → Static colors → Animated blobs → Moving lines

Active Layers:

  • Layer 1: Intense white base (#FCFDFE)
  • Layer 2: Strong vignette (35% dark edges + corner shadows)
  • Layer 3: Subtle static colors (edges only)
  • Layer 4: Gentle animated blobs (edges only)
  • Layer 5: Multi-directional animated lines (5 total)
  • Layer 6: Floating sparks/stars (10 particles)
